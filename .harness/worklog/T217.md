## Implementation notes

- Mirrors `src/jobs/movies/stages/merge.ts` exactly, adapted for TV:
  - Uses `/search/tv` TMDB endpoint (not `/search/movie`)
  - TV results use `name` + `first_air_date` (not `title` + `release_date`)
  - TV TMDB genre ids (10759=Action & Adventure, 10765=Sci-Fi & Fantasy, etc.)
  - Owned set built from `TvSnapshotFile.shows[].tmdbId`
  - Config from `tvRecsConfig` (not `moviesConfig`)
- Reuses `RECS_JOB = 'tv-recs'` + `recKey()` from `src/jobs/tv-recs/recs.ts`
- `ignoreSurfacedItem('tv-recs', recKey(tmdbId))` excludes ignored shows
- Top-up loop fans out over TV `BRANCHES` from `stages/branches.ts` via `collectBranchSuggestions`
- Bounded concurrency via `runBounded` (same pattern as movies)
