# T215 worklog — TV recommender infrastructure libs

## What was built

- `src/jobs/tv-recs/recs.ts` — pure helpers mirroring `movies/recs.ts`:
  `mulberry32`, `seededShuffle`, `primaryGenre`, `stratifiedSample`, `topGenres`,
  `thinGenres`, `creatorsOwnedAtLeast` (TV analog of `directorsOwnedAtLeast`),
  `dedupeRawByTitleYear`, `balanceByGenre`, `mergeLens`, `recKey`, `RECS_JOB`.

  3 serendipity random branches + 5 targeted: creator-completion, top-genre-canon,
  thin-genre, older-era, world-tv. Branch IDs prefixed `tv-rec-*`.
  Exported: `BRANCHES`, `branchById`, `BranchSpec`, `BranchContext`, `BranchKind`,
  `ownedByCreators`, `ownedInGenres`, `ownedPreYear`, `ownedNonAnglophone`,
  `ANGLOPHONE_COUNTRIES`.

- `src/jobs/tv-recs/stages/recommend.ts` — shared branch runner:
  `runBranch`, `collectBranchSuggestions`, `makeBranchJob`, `parseSuggestions`,
  `recentTitles`, `allHistoryTitles`. All Claude calls go through
  `runClaude` from `src/services/claude.ts` (routes via `callService('claude-cli', …)`).

- `src/jobs/tv-recs/types.ts` — added `RecsHistoryFile` interface.

- `src/jobs/tv-recs/recs.test.ts` — pure helper tests: stratified balance, thin
  stratum drain, seed determinism, seededShuffle non-mutation, balanceByGenre cap,
  dedupeRawByTitleYear, topGenres, thinGenres, creatorsOwnedAtLeast.
