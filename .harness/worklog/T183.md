# T183 worklog — per-branch owned-awareness to reduce drop rate

## What was built

### `branches.ts` (core prompt changes)
- Added `alreadySuggested?: string[]` to `BranchContext` — full bounded history, replacing the windowed `recent` list when present.
- Replaced the shared `ownedBlock` (generic stratified sample) with per-lens owned subset helpers:
  - `ownedByDirectors(movies, dirNames, cap)` — for `rec-auteur` (owned films by qualifying directors)
  - `ownedInGenres(movies, genres, cap)` — for `rec-canon` (owned films in top genres) and `rec-thin-genre` (owned films in thin genres)
  - `ownedPreYear(movies, beforeYear, cap)` — for `rec-older-era` (owned pre-1980 films)
  - `ownedNonAnglophone(movies, cap)` — for `rec-world-cinema` (owned non-Anglophone films)
- Random branches still use the stratified sample (appropriate for serendipity; no narrow lens to filter by).
- Replaced `recentBlock` + `excludeBlock` (two separate functions) with `avoidBlock` — a single combined block that merges `alreadySuggested` (or `recent` as fallback) + `exclude` (T162 top-up list). Phrasing is "avoid re-suggesting these specific titles", never "prefer obscure films".
- Exported the lens helpers for unit tests.

### `recommend.ts`
- Added `allHistoryTitles(historyFile, cap)` — loads ALL recommendation history up to `cap`, exported for reuse in `merge.ts`.
- `runBranch()` now loads the full history via `allHistoryTitles` and passes it as `alreadySuggested` to `spec.build()`.
- Logs: "N already-suggested title(s) to avoid (full history, capped at N)".

### `merge.ts`
- Top-up `buildDefaultTopUp()` now imports and uses `allHistoryTitles` to pass `alreadySuggested` in the top-up `BranchContext`, so top-up rounds also avoid previously-recommended films.

### `config.ts`
- Added `recsHistoryContext: 200` (env: `MOVIES_RECS_HISTORY_CONTEXT`) — the cap for history titles passed to branch prompts.

## Tests (`recommend.test.ts`)
Added T183-specific assertions:
- `allHistoryTitles` respects the cap and returns the most recent titles (not full library).
- `alreadySuggested` + `exclude` appear in branch prompts via `avoidBlock`.
- Per-lens helpers (`ownedByDirectors`, `ownedInGenres`, `ownedPreYear`, `ownedNonAnglophone`) return correct subsets, bounded at cap.
- Auteur branch prompt contains director-specific owned films (not a generic sample).
- No "prefer obscure" / "deeper cuts" bias language in prompts.
- All 8 branches accept `alreadySuggested` + `exclude` context without throwing.
