# T234 worklog

## What was wrong

`.panel` in `globals.css` had `border-radius: 10px` but no `overflow: hidden`. The sticky
`thead th` elements inside `.recent-runs-scroll` have `background: var(--panel)` so they painted
over the panel's top border at the corners, making the top-left/top-right border edges invisible.

## Fix

Added `overflow: hidden` to `.panel` in `globals.css` (line 81). This clips any inner content
to the rounded rectangle, so the border is continuous on all four sides regardless of what's
inside — works for all themes × light/dark since it's geometry, not colour.

## Verification

- `npm --prefix dashboard run build` — green
- `node dashboard/scripts/mobile-check.mjs` — all 8 pages pass (0px overflow, 0 boundary crossings)
