# T075: Database page — padding inside Common queries / Browse tables button panels

## What was done

Added `padding: 12` inline to the two `div.panel` elements on `/db` that hold
the Common queries and Browse tables buttons (lines 132 and 163 of
`dashboard/app/db/page.tsx`). These panels used only `display:flex; flexWrap;
gap:8` with no padding, so buttons sat flush against the panel border. The fix
is inline (not a global `.panel` change) so edge-to-edge table panels elsewhere
are unaffected.
