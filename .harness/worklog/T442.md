# T442 — plex-language-fix workflow (scan stage)

## What was built

New public example workflow `src/workflows/plex-language-fix/`:

- `types.ts` — ported Plex/TMDB types + `ShowOrMovieEntry`/`FileEntry` from the scratch
  `plex-language-fix/` scripts at the repo root. `FileStatus` reduced to exactly
  `'change' | 'already-correct' | 'no-match'` — the scratch script's `'ambiguous'` status
  was deleted entirely per the owner's explicit instruction.
- `lib.ts` — ported `fetchSections`/`fetchSectionItems`/`fetchItemDetail`/`fetchAllLeaves`/
  `extractTmdbId`/`lookupLanguageDetail`/`evaluatePart`/`summarize`/`buildCandidateLanguages`,
  reusing `src/core/plex-client.ts`'s `plexGet`/`tmdbGet` and the shared `tmdb` service via
  `callService`. `pickAudioCandidate`'s `ambiguous` branch was removed — every tie now
  resolves via the existing heuristic (original-mix label → channel count → codec quality →
  lowest index) and is returned as a normal candidate. Kept the commentary stoplist and the
  Signs & Songs-only subtitle classifier verbatim (both were validated live and catch real bugs).
- `config.ts` — reuses `PLEX_MOVIE_SECTION`/`PLEX_TV_SECTION` (no duplicate env vars) plus a new
  optional `PLEX_DOWNLOADABLE_SECTION` (unset by default = excluded).
- `contracts.ts` — `plexLanguageScanContract` (shape + non-empty check on the scan JSON). Not
  wired to any DAG edge yet (one member only, no gate needed) — ready for when the planned
  apply/no-track-flag stages join.
- `stages/scan.job.ts` + `stages/scan.ts` + `stages/scan.test.ts` — the read-only scan stage.
  Scans configured sections, evaluates every file, writes `data/out/language-scan.json`. Verbose
  per-title/per-section logging. Idempotent per ISO calendar week via the `work_items` ledger
  (`weekKey`, mirroring `plex-space-saver`), `detail: { format: 'json', path }` (T262 convention).
- `plex-language-fix.workflow.ts` — `category: 'regular-maintenance'`, `schedule: '0 4 * * 0'`
  (Sunday 04:00, distinct from `plex-space-saver`'s 06:00 and `missing-tv-seasons`'s Monday 09:00),
  single member `plex-language-scan`, left easy to extend with `dependsOn` for future stages.
- `CLAUDE.md` — full current-state documentation for the folder.

Root `CLAUDE.md`: added a `plex-language-fix` row to the "Shipped example workflows" table
(13 → 14 workflows), updated the shared Plex/TMDB connectivity note (4 → 5 workflows), updated the
file map's per-workflow-folder list and the "public example workflows" list.

`.env.example`: documented `PLEX_DOWNLOADABLE_SECTION` (commented out, unset by default) next to
the existing `PLEX_MOVIE_SECTION`/`PLEX_TV_SECTION` entries.
