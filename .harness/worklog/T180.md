# T180 worklog

## What was done

Added `justify-content: center` to `.dag` in `dashboard/app/globals.css` (line 153).

The `.dag` container is a flex row (`display: flex; flex-wrap: wrap`). Adding `justify-content: center` horizontally centers the wave columns when the total content width is less than the container width. When the graph fills or exceeds the container, `flex-wrap: wrap` still takes over naturally — no change in behavior.

On mobile, `.dag` is overridden to `flex-direction: column; align-items: stretch`, so `justify-content` there is irrelevant (column direction applies it to the cross axis which is already stretch).
