# T021 worklog

## Attempt 1 — 2026-06-21

### What was done

Added clickable filtering to the four overview stat tiles on the dashboard Overview page.

**Changes:**
- `dashboard/app/page.tsx`: Added `activeFilter` state (`'running' | 'success' | 'failed' | 'stuck' | null`). Converted each `<div className="statcard">` to a `<button>` with an `onClick` that toggles the filter (clicking the active tile again clears it). Added a filter indicator line with a "× clear" button when a filter is active. Computed `visiblePipelines`, `visiblePipelineRuns`, and `visibleStuck` derived arrays that are filtered based on `activeFilter`, and used these in the three list sections below the tiles.
- `dashboard/app/globals.css`: Added `cursor: pointer`, `text-align: left`, `font: inherit`, `color: inherit`, hover + active (`.active`) highlight styles for `.statcard` button elements.

**Filter mapping:**
- Running → show pipelines with `last_run.status === 'running'`; pipeline runs with `status === 'running'`; stuck items hidden
- Succeeded → pipelines + pipeline runs with `status === 'success'`; stuck hidden
- Failed → pipelines + runs with `status in [failed, partial, cancelled]`; stuck hidden
- Stuck → stuck items shown, pipeline runs unfiltered, pipeline cards with `p.stuck > 0`
