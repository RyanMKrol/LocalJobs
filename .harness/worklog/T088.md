# T088 worklog — Gate detail: render code/JSON-shaped values in a clear code block

## Attempt 1 — 2026-06-22

### What was done

Added a reusable `.code-block` CSS class to `dashboard/app/globals.css` and updated
`dashboard/app/workflow-runs/[id]/gates/[producer]/[key]/page.tsx` to use it for the
three code-shaped values on the gate detail page:

1. **Artifact format** (`shape.format`) — changed from `<p className="muted mono gate-format">` to
   `<code className="code-block gate-format">`. This renders strings like `{ perfumeId: fragranticaUrl }`
   in a visually distinct block instead of inline prose.

2. **Per-expectation actual values** (`r.actual.actual`) — changed from
   `<span className="muted mono gate-expect-actual">actual: …</span>` to
   `<code className="code-block gate-expect-actual">…</code>`. Dropped the inline "actual:" prefix
   since the block is visually self-contained.

3. **Sample** (`result.sample`) — restructured from a single `<p>` with inline `<span className="mono">`
   to a `<div className="gate-sample">` containing a `<p className="gate-sample-label muted">` label
   and a `<code className="code-block">` for the sample text.

### CSS changes

- Added `.code-block`: mono font, 12px, `var(--bg)` background, `var(--border)` border, 4px radius,
  `6px 10px` padding, `pre-wrap` + `overflow-wrap: anywhere` for long content.
- `.gate-format`: removed `font-size: 12px` (comes from `.code-block` now).
- `.gate-expect-actual`: changed from `font-size: 12px` to `margin-top: 3px` for spacing.
- `.gate-sample`: removed `font-size: 12px`; added `.gate-sample-label` for the "What actually flowed:" label.
