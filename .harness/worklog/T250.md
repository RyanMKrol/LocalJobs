# T250 — Symmetric vertical padding in DAG graph container

## What was done

Increased `fitViewOptions.padding` in `dashboard/app/components/DagFlow.tsx` from `0.15` to `0.25`.
React Flow's `fitView` applies this padding uniformly on all four sides when zooming to fit the
graph bounding box. At 0.15 the top/bottom margins were visually unequal (graph flush to the top,
sliver at the bottom). Bumping to 0.25 gives React Flow more room to breathe and the result is
symmetric vertical centering.

No CSS changes were needed — adding `padding` to the `.dag-flow-wrap` container would shrink the
React Flow canvas area inside a fixed-height div, which doesn't help with centering.
