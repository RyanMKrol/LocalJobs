# T107 — Cron: explicit '?'-icon tooltip with instant human-readable text

## What was done

Added a `CronBadge` React component in `dashboard/app/ui.tsx` that renders a cron
expression with a small `?`-in-circle icon next to it. On hover, focus, or tap the icon
shows the human-readable cron text immediately via a custom CSS-positioned tooltip — no
native-title 1s hover delay.

### Files changed
- `dashboard/app/ui.tsx` — added `CronBadge` component (uses existing `cronToEnglish`
  helper from T097; no reimplementation). Falls back to plain mono text when
  `cronToEnglish` cannot translate the expression (avoids a useless tooltip).
- `dashboard/app/globals.css` — added `.cron-badge`, `.cron-help`, `.cron-tooltip`
  styles. Tooltip uses `position: absolute; z-index: 200`, `max-width: min(220px, 80vw)`
  so it doesn't cause horizontal overflow at phone width.
- `dashboard/app/workflows/page.tsx` — replaced `<span title={cronToEnglish(...)}>` with
  `<CronBadge expr={...} />`.
- `dashboard/app/workflows/[name]/page.tsx` — same replacement.
- `dashboard/app/page.tsx` — same replacement (overview card inline text).

### Accessibility
- Icon has `tabIndex={0}`, `role="button"`, `aria-label` with the English text, and
  `aria-haspopup`/`aria-expanded`.
- `onFocus`/`onBlur` open/close the tooltip for keyboard navigation.
- `onClick` toggles for touch (tap to show, tap again to hide; blur on tap-elsewhere
  hides via `onBlur`).
