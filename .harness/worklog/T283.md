## What changed

- Added `taskById` (a `Map` built from the full `tasks` array) so a dep id can be resolved to its
  own task record (in particular its `gate`).

  fixture, now a standalone Ready example with no dependents.

  `#task-T098` to become visible (T098 renders under "Needs a human", already `<details open>`),

  T050 in Done) are unchanged.
