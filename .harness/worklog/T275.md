## What changed
`dashboard/app/services/page.tsx`:
- Imported `Pill` from `../components/Pill`.
- Replaced the ad-hoc `<span className="pill paid">`/`<span className="pill free">` service-billing

- Replaced the ad-hoc `<span className="pill" title="...">` "edited" badge (shown when

No RunButton usage applies to this page — it has no workflow-run trigger, only a consumers-modal
"Close" button and per-row "Edit limits"/"Save"/"Cancel" buttons, none of which match `RunButton`'s
purpose (workflow run/running state). Left those as plain `<button className="btn ...">` since T268
only defined `RunButton` and `Pill` as reusable so far, and swapping unrelated buttons would be
scope creep beyond "like-for-like using existing shared components".
