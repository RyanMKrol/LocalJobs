### Mobile fix
Mobile check was failing because React Flow internal elements (`react-flow__nodes`, `react-flow__node`) have `overflow: visible` and spill their containers on narrow viewports. Added `overflow: hidden` to those selectors inside `.dag-flow-wrap` in the `@media (max-width: 640px)` block. The graph is still visually correct — the parent container already clips via `overflow: hidden`.
