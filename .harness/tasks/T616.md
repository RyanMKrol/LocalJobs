## Overview
The workflow-detail Output section only shows a clickable 'View' button for produced items that recorded a markdown artifact; items using the newer T262 output form (detail.format:'json' + detail.path) get no button and can't be opened — even though the backend already serves them. Extend the viewability signal so JSON/path outputs are clickable too. Affects overrides-audit, plex-space-saver, and plex-language-fix.
