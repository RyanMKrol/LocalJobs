collection lists the films you're MISSING, but it's not always obvious which film(s) in that
collection you already OWN — so it's hard to recognise the franchise at a glance. For each
collection with gaps, also show **one example film you already own** from that collection, as a
recognisable anchor.

- The franchise-gap detection already knows, per collection, the released parts and which are owned

  type in `src/jobs/movies/types.ts`) as needed to carry the owned example, and update the report
  renderer (the franchise-gaps stage / notify stage that writes `franchise-gaps.md`).

  landed, apply this to that same report.
