## Overview

The dashboard's markdown popovers (used for `stocks-sync`'s `portfolio.md` digest and every other
workflow output preview) render GitHub-Flavored-Markdown pipe tables as raw, unrendered `| a | b |`
text instead of an actual `<table>`. Root cause: `react-markdown` (`dashboard/package.json`) only
supports core CommonMark out of the box — pipe tables (plus strikethrough/autolinks/task lists) are a
GFM extension that requires explicitly wiring the separate `remark-gfm` plugin, which isn't installed
anywhere in this repo. Install `remark-gfm` and pass it to all four `<ReactMarkdown>` call sites.

   T115 already documented ("no rehype-raw — raw HTML is escaped, not executed"). This task changes
