## Overview
Give the missing-tv-seasons workflow's first two stages real work_items ledger rows (one per Plex show, chained by rootKey) so the dashboard's per-run Inputs & Outputs panel shows real data for them instead of being blank — today only stage 3 (plex-seasons-notify) records any ledger rows at all. A show with no tmdb:// GUID gets a deliberate status='failed' row that reuses the existing dashboard stuck-item infrastructure (Unstick/Ignore), at the accepted cost that it also blocks the whole weekly run until a human resolves it.

## Do
