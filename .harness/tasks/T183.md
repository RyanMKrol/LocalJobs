## Do

Reduce the recommender's high **drop rate** — most branch suggestions get dropped as "already OWNED"
or "already recommended/ignored", which forces many slow top-up rounds — by giving each branch
**owned-awareness** in its prompt. **Important constraint: do NOT bias branches toward obscurity /
deeper cuts.** The owner explicitly WANTS to be told about acclaimed/canonical films they're missing,
so the goal is "don't re-suggest these *specific* owned/already-suggested titles", NOT "prefer
lesser-known films". A canonical film the owner is missing must still be recommended.

Specifically:
1. **Per-lens owned subset (not the whole library).** Give each branch a TARGETED subset of the
   owner's owned titles relevant to that branch's lens — never the full ~1,537-title library. E.g.
   the world-cinema branch sees owned world/foreign films; older-era sees owned older films;
   top-genre-canon / thin-genre see owned films in the relevant genres; auteur sees owned films by
   the directors it's considering. Compute these from the snapshot (genres, year, directors) / taste
   profile. For the serendipity/random branches (no narrow lens), pass a representative sample and/or
   lean on the already-suggested set below rather than a full dump.
2. **Already-suggested context.** Pass the run's already-collected picks + the `movie-recs` history
   (already-recommended / ignored) so branches don't re-suggest within or across rounds — this kills
   the "already recommended/ignored" drops.
3. **No obscurity bias (the constraint above).** Phrase it as "avoid re-suggesting these specific
   titles you already own / that were already suggested", never "prefer obscure films".
4. **Keep the deterministic filter as the source of truth.** The prompt context is a best-effort
   hint to reduce waste; `rec-merge`'s owned/dedup/quality filter still runs and stays authoritative
   (the LLM won't perfectly honor an exclusion list, so the filter must remain).
5. **Keep prompts bounded** (token-sensible) — targeted subsets, not the full library; log how many
   owned / already-suggested titles were passed to each branch.

Goal: each branch's suggestions LAND more often → fewer drops → fewer top-up rounds (complements
T182's parallel rounds), with **no change to WHAT qualifies** as a recommendation (missing canon
still surfaces).
